Ubirch Certificate Reference

Here you can find a collection of different examples of creating UBIRCH Protocol Package (UPP) certificates and verifying them.

Currently the following types of UPPs can be created by the @ubirch/ubirch-certification-js ( https://www.npmjs.com/package/@ubirch/ubirch-certification-js):

  • SIGNED

To execute the examples You need to contact the UBIRCH Support support@ubirch.com to get

  • client certificate
  • identity UUID (deviceId)
  • verification access token
  • stage (normally stage for tests will be DEMO)

    • Simplest example for creation of a signed UPP.

      To execute this example you need to add your device and stage directly to the script inside of /pages/simple-cert.html

      ubirchCertification = new window.UbirchCertification({
    deviceId: '<YOUR_DEVICE_UUID>',
    stage: '<USED_STAGE>'
});

    • Example for creation of a signed UPP and verifying it

      To execute this example you need to add your device, token and stage to environment.ts

    • Example for creation of a signed UPP and verifying it

      This is the live demo from documentation pages.

      In this example you can add your deviceId, verification token and stage directly into the form. After setup press 'Setup UBIRCH certification and verification' and insert data for certification OR verification.
