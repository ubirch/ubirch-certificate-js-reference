Ubirch Certificate Reference

Here you can find a collection of different examples of creating UBIRCH Protocol Packages (UPP) certificates and verifying them.

Currently the following types of UPPs can be created by the [@ubirch/ubirch-certification-js](https://www.npmjs.com/package/@ubirch/ubirch-certification-js):

  • SIGNED

  • Simplest example for creation of a signed UPP.

    To execute this example you need to add your device and stage directly to the script inside of /pages/simple-cert.html

    ubirchCertification = new window.UbirchCertification({
    deviceId: '<YOUR_DEVICE_UUID>',
    stage: '<USED_STAGE>'
});

  • Example for creation of a signed UPP and verifying it

    To execute this example you need to add your device, token and stage to environment.ts
